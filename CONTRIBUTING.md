# Contributing to WhereTheWaterFlows (WWF)

We welcome contributions to WWF:
- file issues for bugs or missing documentation
- make pull requests for fixes and extensions

If you plan to implement a larger feature, we encourage you to file an issue first to
discuss your plan and to see if it is in scope for WWF.

## Scope of WWF

WWF aims to be reasonably narrowly focused on waterflow and other transport via routing
algorithms. Extra functionality, such as the functionality implemented in `Randomly` and
`Subglacially`, may be added to submodules.

However, we encourage build new packages based on top of WWF; an example is
[SUGSET-2D](https://bitbucket.org/IanDelaney/sugset.jl/src/id-2d/) for sediment transport
through glaciers.

## AI usage

AI tools, such as LLMs, are used to create content for this repository.
Please disclose AI usage in your contributions (pull requests but also issues generated by AI).
Contributors must thoroughly review AI generated content and remain fully
responsible for the content they submit.
