# Contributing to WhereTheWaterFlows

We welcome contributions to WWF:
- file issues for bugs or missing documentation
- make pull requests for fixes and extentions

If you plan to implement a larger feature, we encourage you to file an issue first to
discuss your plan and to see if it is in scope for WWF.

## Scope of WWF

WWF aims to be reasonably narrowly focused on waterflow and other transport via routing
algorithms. Extra functionality, such as the implemented in `Randomly` and `Subglacially`, may be added
to submodules.

However, we also encourage to build new packages, an example is
[SUGSET-2D](https://bitbucket.org/IanDelaney/sugset.jl/src/id-2d/) for sediment transport
through glaciers, based on top of WWF.
